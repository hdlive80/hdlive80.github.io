Typography

I have chosen to use the font Montserrat through out my entire website. When used throughout it fulfills the design principle of consistency. I also felt this was a good font because it is a Sans Serif font so it is clean a precise. This is the same characteristics that someone would expect when they are in search for the weather forecast.
